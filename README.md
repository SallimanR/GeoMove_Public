# GeoMove public repository

### For maps see https://www.npmjs.com/package/@geomove/maps and [docs](./frontend/packages/maps/README.md)
### For geo utilities see https://www.npmjs.com/package/@geomove/go and [docs](./frontend/packages/geo/README.md)

### geo APIs:
- https://geomove.online/style/style/style.json - maps style.json
- https://geomove.online/tiles - PMTiles API (tiles for maps)
- https://geomove.online/geocoding - Search and severse geocoding API
- https://geomove.online/routing - Routing and map matching API
